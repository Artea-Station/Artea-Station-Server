modular_skyrat/modules/alerts/code/priority_announce.dm

/proc/priority_announce Sends an announcement to all players and formats it accordingly. Use this for big bad shit.
/proc/print_command_report If the announcer overrides alert messages, use that message.
/proc/minor_announce Sends a minor annoucement to players. Minor announcements are large text, with the title in red and message in white. Only mobs that can hear can see the announcements.
/proc/call_emergency_meeting Summon the crew for an emergency meeting
